# TypeMoor

TypeMoor turns CBT reframing into a typing drill. Caught in a loop of layoff dread or AI-replacement fear? Type carefully worded cognitive reframes until accuracy and rhythm crowd out the spiral. Each session logs your speed and accuracy on-device — measurable progress at a skill that also quiets runaway thinking.

iPhone & iPad. Free. No ads, no in-app purchases, no accounts, no network.
